# PeakFuel Landing Page

A modern React + Vite landing page for gym nutritional products.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Scripts

- `npm run dev` - start the local dev server
- `npm run build` - create a production build
- `npm run preview` - preview the production build locally

## Notes

The page is structured for conversion with a strong hero, product cards, trust signals, testimonials, FAQ, and a final call to action.
